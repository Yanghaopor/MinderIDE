# MinderIDE
基于https://github.com/Yanghaopor/ITSQL_Backend_framework框架的AI智能IDE，现在只属于测试版本

#介绍
此IDE当前只能生成对应的语言文件，可以自行运行
其基本原理依靠Dify进行
可自行搭建Dify的API其具体为IDEs.yml
而且API接口前往文件路径下的MUB.json更改API路径“AIurl”

其中yun.zip为源代码
MinderIDE.7z为打包好的软件，请安装到无中文路径下，且为管理员运行

#运行环境
需要python才能完好运行
python未封装
